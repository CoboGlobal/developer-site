The Cobo Payment Solution currently supports two major stablecoins, USDC and USDT, and can be used on multiple mainstream blockchains. The detailed support is as follows:


## USDC supported chains
* Ethereum (ETH_USDC)
* Arbitrum (ARBITRUM_USDCOIN)
* Base (BASE_USDC)
* Polygon (MATIC_USDC2)
* BNB Smart Chain (BSC_USDC)
* Solana (SOL_USDC)

## USDT supported chains
* TRON (TRON_USDT)
* Ethereum (ETH_USDT)
* Arbitrum (ARBITRUM_USDT)
* Base (BASE_USDT)
* Polygon (MATIC_USDT)
* BNB Smart Chain (BSC_USDT)
* Solana (SOL_USDT)

## Use TTRON for testing

Besides, you can also use TTRON (TRON testnet) for testing in the development environment. Cobo Payment Solution currently supports test coin TTRON_USDT1:

* Token contract address: `TXYZopYRdj2D9XRtbG411XZZ3kM5VkAeBf`
