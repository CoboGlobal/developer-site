#### **July 17, 2024**

This is the first release of the Cobo WaaS 2.0 API.

This release includes support for [Custodial Wallets](https://manuals.cobo.com/en/portal/custodial-wallets/introduction) and [MPC Wallets](https://manuals.cobo.com/en/portal/mpc-wallets/introduction), with the following categories of operations:

- [General wallet operations](/developers/v2/api-references/wallets/list-supported-chains)
- [MPC Wallets-specific operations](/developers/v2/api-references/wallets--mpc-wallets/list-all-cobo-key-share-holders)
- [Transactions operations](/developers/v2/api-references/transactions/estimate-transaction-fee)
- [Webhook operations](/developers/v2/api-references/developers--webhooks/list-webhook-endpoints)
