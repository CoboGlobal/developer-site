本指南将帮助您通过 Payment Management App 快速开始使用充值模式，包括创建付款方、获取充值地址、查看余额和结算操作。

您也可以通过 API 来完成本文涉及的所有操作。详情可参考[获取充值地址](/v2_cn/payments/get-topup-addresses)和[结算](/v2_cn/payments/settle-topup)。

<Note>
  本指南以开发环境为例。在生产环境进行操作时，请将 URL 等参数替换为生产环境的对应值。
</Note>

## 前提条件

您已完成[前置准备](/v2_cn/payments/preparation)中提到的所有步骤。

## 操作步骤

### 1. 创建付款方

1. 登录 Cobo Portal [开发环境](https://portal.dev.cobo.com/login)。
2. 在左侧导航栏中点击 **Apps**，然后点击 **Payment Management** 卡片，启动 App。
3. 在 App 的左侧导航栏中点击**充值**。
4. 点击右上角的**创建付款方**按钮。
5. 在弹出的表单中：
   - 选择该付款方关联的商户。如果您是商户（直接服务付款方），则可关联您在[前置准备](/v2_cn/payments/preparation)中为自己创建的商户。
   - 选择代币和网络。Cobo 会为该付款方创建一个该网络上的专属充值地址。创建付款方后，您还可以为该付款方在其他网络上创建更多充值地址，以支持多链充值。
   - 输入自定义付款方 ID。此 ID 需要是此付款方在您的系统中的唯一标识。
5. 点击**创建**完成付款方创建。

<img src="/v2_cn/images/payments/create-payers.png" className="screenshot_full_screen" alt="创建付款方" />

### 2. 获取充值地址


1. 在付款方列表中找到目标付款方，然后点击右侧的**查看地址**按钮。

<img src="/v2_cn/images/payments/view-address-button.png" className="screenshot_full_screen" alt="查看充值地址" />
2. 您可以在地址列表中查看已创建好的地址。如果您想要添加其他公链上的地址，可以点击**创建地址**按钮进行创建。
<img src="/v2_cn/images/payments/view-topup-addresses.png" className="screenshot_full_screen" alt="查看充值地址" />
3. 将所需充值地址提供给付款方，完成支付。


### 3. 查看余额

在结算前，您可以在**余额**页面查看各个维度的余额信息。

1. 在左侧导航栏中点击**余额**。
2. 在右上角的筛选框中选择余额类型：
   - **商户余额** - **充值模式**：查看各个商户在充值模式下的余额
   - **开发者余额**：查看您作为开发者的所有余额，包括从商户收取的手续费等
3. 筛选后，您可以查看总余额以及每个代币在不同链上的余额。

关于各类余额的具体定义和计算方式，请参考[金额定义与余额计算说明](/v2_cn/payments/amounts-and-balances)。

<img src="/v2_cn/images/payments/view-topup-balances.png" className="screenshot_full_screen" alt="查看订单余额" />

### 4. 结算余额

1. 在左侧导航栏中点击**结算**。
2. 选择您想要结算的余额类型以及结算方式。在本例中，我们选择通过加密货币的方式结算商户在充值模式下的余额。

<img src="/v2_cn/images/payments/settle-topup-button.png" className="screenshot_full_screen" alt="结算订单入口" />

3. 填写结算信息：
   - 选择要提现的代币和网络。
   - 选择要结算的金额。
   - 输入接收地址。<Note>您需要联系 Cobo 销售团队录入相应的结算账户信息，该地址才会出现在下拉列表中。</Note>
   - 添加可选的描述信息，如结算备注等。
   - 按照页面提示确认并提交结算信息。

<img src="/v2_cn/images/payments/settle-topup-form.png" className="screenshot_modal" alt="结算订单表单" />
